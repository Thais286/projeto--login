Projeto Tela de Login Responsiva

Este projeto é uma tela de login responsiva desenvolvida como parte de um exercício de estudo.

**Base de Estudo:** Este trabalho é inspirado no projeto prático de responsividade do **Curso em Vídeo**, ministrado pelo Professor **Gustavo Guanabara** (Módulo 4, Capítulo 26).

## ✨ Funcionalidades Principais

* **Layout Adaptável (Responsivo):** A tela se ajusta automaticamente para celular, tablet e desktop.
* **Design Horizontal:** Uso de Flexbox para posicionar o formulário de login e a imagem lado a lado em telas maiores.
* **Componentes:** Formulário de login/senha e link para recuperação de senha.
* **Ícones:** Utilização da biblioteca Material Symbols (Google Fonts) nos campos de entrada.
